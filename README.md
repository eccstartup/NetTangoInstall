# NetTango 本地安装

### 分两步
* run [nettango-install.sh](https://github.com/eccstartup/NetTangoInstall/blob/master/nettango-install.sh)
* Visit http://localhost:9000/nettango-builder

### 安装软件

* 安装`sbt`
* 安装`node`
* 安装`jdk 8`

请自行安装
